# GenAI Launchpad Chat example

### Getting Started

1. Clone the repository

```bash
git clone git@github.com:datalumina/genai-launchpad-chat-example-frontend.git 
```

2. Install dependencies

```bash
npm install 
```

3. Set up environment variables
   Create a .env file in the root of the project. Copy and paste the variables from the .env.example and update the
   values.

4. Run the development server

```bash
npm run dev 
```

Open http://localhost:3000 in your browser to see the application.
